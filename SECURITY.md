# Security Policy

### Supported Versions

| Version | Supported |
|---------|-----------|
| latest (`main`) | ✅ |
| older releases  | ❌ |

## Reporting a Vulnerability

**Не создавай публичный issue или discussion** — это раскрывает уязвимость до того как она исправлена.

Напиши через GitHub: **[Private vulnerability reporting](../../security/advisories/new)**

Включи в сообщение:
- Описание уязвимости и потенциальный impact
- Шаги для воспроизведения
- Версию / коммит где обнаружено

**Ответ в течение 48 часов. Исправление — в течение 14 дней** (критические CVE — быстрее).
